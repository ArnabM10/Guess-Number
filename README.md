# Guess the Number Game

This is a simple "Guess the Number" game built using HTML, CSS, and JavaScript. The user tries to guess a randomly generated number between 0 and 100. The game provides feedback on whether the guess is too low, too high, or correct. The game includes a "Start New Game" button to reset the game.

## Demo

Check out the live demo of the game [here](https://guess-number-arnab.vercel.app/).

## Features

- Random number generation between 0 and 100.
- Feedback for too low, too high, or correct guesses.
- Displays the remaining number of guesses.
- "Start New Game" button to reset the game.
- Responsive and styled UI.

## Installation

1. Clone the repository:
    ```bash
    git clone https://github.com/ArnabM10/Guess-Number.git
    ```
2. Navigate to the project directory:
    ```bash
    cd guess-the-number-game
    ```

## Usage

1. Open the `index.html` file in your web browser:
    ```bash
    open index.html
    ```
2. Enter a number in the input field and click "Guess" to see if you can guess the randomly generated number.
3. The game will provide feedback on whether your guess is too low, too high, or correct.
4. If you run out of guesses or guess the correct number, you can start a new game by clicking the "Start New Game" button.
